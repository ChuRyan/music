Hector the Hippo

Hector the Hippo is an educational game built during the Cupertino Library's 2016 Summer Hackathon, winning the first prize. The goal of the game is to sort waste into the right bins; after two or three playthroughs, players will be able to have a great understanding about where to place their trash. Four different categories are used in this game: Trash, Recyclables, Yard Waste/Compost, and Special Cases. This game was built using a coding language similar to JavaScript. The graphics and animations were created in Blender 3D.
